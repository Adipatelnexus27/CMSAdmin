# CMS Admin Portal

Enterprise Admin Portal for Claim Management System (CMS), built with React, TypeScript, Material UI, and Vite.

## Features
- Full claims lifecycle modules (registration to payment and audit)
- Role-ready route guarding and auth context
- Shared app shell with enterprise navigation
- API client abstraction for ASP.NET Core backend integration
- Reusable data table and dashboard components

## Prerequisites
- Node.js 20+
- npm 10+

## Run
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```

## Folder Structure
```text
src/
  app/
  core/
```
